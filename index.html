svgfighter by sminnee

Simple space fighter game implemented in SVG and JavaScript

When SVG support was first added to browsers, I got all excited about the possibilities and made this simple space ship fighting game. At a high level, the interface is inspired by Star Control.

Dependencies

An SVG-aware browser.

Install

* Download * Open fighter.svg in your browser * Fight! ** Blue player: Arrow keys to steer, and Enter to shoot ** Green player: W, A, S, D to steer, and Control to shoot
